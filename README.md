## Language Features
Instruction & Meaning:

- ```SHALLOW```:	Move pointer left
- ```DEEPER```:	  Move pointer right
- ```REEL```:	    Increment current cell
- ```SLACK```:	  Decrement current cell
- ```SOUND```:	  Output current cell as number
- ```SURFACE```:	Output current cell as ASCII
- ```DROP```:	    Read integer input into cell
- ```DIVE```:	    Loop start
- ```RISE```:	    Loop end
- ```ABORT```:	  Halt program
